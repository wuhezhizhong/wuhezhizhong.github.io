文件删除代码：
利用java在已建好的文本框中写入文字：
package alvin.alvinio;

import java.io.*;

public class WriteText {

public static void writeText(String path, String text) {
try {
FileOutputStream o = new FileOutputStream(path);
o.write(text.getBytes(“GBK”));
o.close();
} catch(Exception e) {}
}

//测试函数
public static void main(String[] args) {
//将指定的文本内容以 GBK 编码写入磁盘的指定路径。
String content = “在这里写入想要写入TXT中的文字”;
WriteText.writeText(“C:/hello.txt”, content);
}
}

人类简史
date: 2017-04-07 16:22:26
tags:欢迎来到每次看心情更新的博客，大家好，我是脑壳疼。
这次呢，我开始写人类简史的笔记了。
第一章：人类
这章呢主要介绍在七万年前，一些属于“智人”这种生物开始创造出复杂的架构，此架构呢成为文化。
而架构继续发展就成为了历史学。
而历史学的路上呢，有三大重要革命：大约七万年前的“认知革命”让历史正式启动。大约12000年前
“农业革命”让历史发展迅速，也改变了人类的进化方向。大约500年前的“科学革命”也就是人类新创
局。
人类早在250万年前就存在，但是由于世世代代的生存，与其他的生物也并没有太大的区别，然而就是
慢慢的进化才让人类试与天齐，在那个时候，人类也分很多种，就像猩猩和狒狒一样，区别不太大，但是
却是两个种类。
最早的人类从250万年前的东非开始演化，200万年前，有一些远古人类开始踏上路途，足迹遍及北非，
欧洲与亚洲，由于各地的天气，环境不一样，所以人类开始往不同的方向开始发展。在欧洲和西亚的人类
成为了“尼安德特”人。这种人种更为魁梧，肌肉也特别发达，非常适应西方的欧亚大陆，他们贼NB，生
存了大概两百万年。另一类人则称为“梭罗人”他们都是侏儒，高大的人都饿死失去生长优势，经过几代
之后，在弗洛里斯多人都成了小矮人（莫非是霍比特人的祖先？），这些人身高最高不过1米，体重最重
也不过25公斤。
思考的代价：
人与动物最大的区别在于人类会思考，人类深深的迷恋着自己的高智能，但是为什么动物就不会思考？
原因在于庞大的大脑也是一种负担，大脑结构脆弱，不利于活动，并且大脑消耗的能量太多，消耗比占25%
，其他猿类只占了8%。于是代价就来了，人类得花更多的时间去寻找食物，其次就是肌肉的萎缩。虽然猩
猩不及人类聪明，但是猩猩却轻松弄死人类，你跟他讲道理有用吗？不过呢，人类大脑思考的好处却显而
易见，汽车，飞机，枪炮，高科技，让我们站在世界的顶端。
厨师的种族：
在踏上食物链顶端的路上，使用火是迈出了一大步，早在80万年前，人类就部分会使用火，在30万年前
火已经用到了炉火纯青的地步。火可以作为光源和热源，也可以抵挡住狮子的攻击。甚至人类还会刻意焚
烧周围环境捕捉猎物，从大火中得到食物，开路，让森林变成原野。
守护者：
虽然火带来许多优势，但是15万年前，人类只是边缘生物，这时他们只能够吓走狮子，在寒冷的夜里取
暖，在有火以后，智人的牙齿和颌骨比祖先小，而脑容量又比较大。大约七万年前，智人从东非扩张到阿
拉伯半岛并且很快席卷整个欧亚大陆。智人来到阿拉伯半岛的时候，已经有许多的其他人，那么其他人后
来怎么样了？一种是混合繁衍，智人来到尼安德特人的地盘时，两个人种互通繁衍，知道合二为一。然而
另外一种说法就是水火不容，要么智人就直接抢占了尼安德特人的地盘，双方总有一方赢得领地。结果还
在有待研究。但是最后活下来的却是智人。在大约3万年前智人已经习惯自己是唯一的人种，很耐接受其
他的可能性。也许他们就是让尼安德特人灭种的呢？

好了今天就到这，下次看心情更新，拜拜。

title: 电影《驴得水观后感》
date: 2017-03-15 18:08:55
tags:—

第一次写博客，请多多指教。
今天说一说电影《驴得水》的感想。
故事发生在1942年的农村，一个缺水的地方，有个学校养了一头驴挑水，可谁都不愿意出养驴的钱，于是校长便将这头驴虚报成了一位名叫“吕得水”的教师，用“吕得水老师”的工资来养驴；当上级领导来检查，要见这位吕老师的时候，大家只能编造各种借口搪塞。最后演发到不可收拾的地步。
故事的主人公为：校长，周铁男老师，张一曼女老师，裴魁山老师，以及特派员，特派员副官，校长的女儿孙佳，铜匠，资助人美国佬。
故事的开始是校长与几个老师开会商量着下学期学生的流量得失，然而这个学期放假之前至考试却只有6个人来上课，这让校长很苦恼，于是校长准备以发放奖学金的方式来鼓励学生们上学。这样他的学校才可以保住，以及坚持他的梦想。
由于这个地方缺水，只能去十几里的地方去拉水来吃，然而谁也没钱去养活这头驴，无奈，校长只能谎报教师人数，说还有一个吕得水老师，其实这个老师就是这头驴。后来核对了一下账单，却没有多少钱可以用了。其中有裴魁山的假牙费，铁男的健身器材费，一曼的布料费。这时校长书“说了多少次了不要用驴得水的费用。”刚说完，一曼说出了校长的眼睛更换费用，当场打校长的脸。后来驴棚着火了。他们几个疯狂的去救火，铁男拿着扫帚去灭火，扫帚烧着了，一曼去驴棚里灭火，这时，裴魁山却过来往一曼身上铲土，（因为没有看到一曼）这时校长拿着被子过来了，说“别打了，一切由我指挥”，然后驴棚就烧的更大了。他们只能眼睁睁地看着驴棚烧毁。
刚好孙佳赶着驴带着水回来了，用吃的水去灭火。校长几个都去极力阻止。这件事就到此告一段落。
后来听说特派员要来，然而，前几次特派员来视察时，他们都说吕得水老师去出差了，然而这次特派员是特地来视察驴得水老师的，还要听驴得水老师讲课。他们无奈，只能找铜匠，给他培训，给他打扮，教他说几句英语，来糊弄特派员。第二天特派员来了，一曼试探着说了几句英语，发现特派员根本不懂英语，于是校长就让一曼当翻译，铜匠来说本土的方言，结果就这么糊弄这过去了。期间裴魁山像张一曼示爱，表示不在意一曼的过去（一曼的生活特别放荡，跟裴魁山等睡过。），但是一曼说“我只想过这样的生活，我也没想到你会往哪方面想，以后我不会再招你了”。
终于特派员糊弄过去了，但是要拍照，由于铜匠有迷信，说拍照会死人的，死活不肯拍，这时，只能由一曼出马，就是跟铜匠睡了，铜匠才肯同意拍照。并从此爱上了张一曼。期间裴魁山发现后，伤心的离开了。铜匠走的时候校长跟他说“有教无类”并给了他几本书，说他天赋异丙可以多看看书。后来特派员指出由于铜匠的英语（方言）说的特别溜，所以美国特地资助了驴得水老师3万美金（这3万都由校长，一曼，铁男，魁山瓜分。），伺候他们过上了特别快活的生活。
由于铜匠有老婆，铜匠回家两个月，天天捧着本小破书看，活也不干了，感觉他变了，并且姿势都变了，所以铜匠老婆知道，有人跟铜匠搞破鞋了。由于这事这么闹着也不是办法，裴魁山就说，谁干的谁心里有数，别干了不承认。校长也劝说一曼，要顾全大局。所以一曼气不过就去承认了跟铜匠高破鞋。铜匠老婆要收拾一曼的时候，铜匠拦了下来，从来不敢跟自己大小声的铜匠竟然敢拦自己，并且叫自己滚，他老婆呆住了。她从来没有见过铜匠敢这样。这时一曼忍不住说话了，对铜匠说“我睡你只不过是为了你去拍照，没有别的意思，在我眼里你就是一个牲口”铜匠听了很震惊，失落的走了。这时，张一曼流着眼泪说了声“对不起”
后来特派员带了一个美国人过来，此人就是资助驴得水老师的美国人，然而此人非要见驴得水一面，以保证，他们不是在骗自己。然而铜匠消失了一个多月谁也找不到，特派员要求必须找到，这时，铜匠刚好来了，所有人都把他当宝似的捧着，这时，铜匠的报复就开始了，他提出了许多的要求，例如，要求在场的人骂张一曼，这时铁男站出来说谁敢骂一个试试，由于特派员护着，所以铜匠背也直了许多，指出如果不骂一曼，他就揭穿他们的骗局。无奈他们只能顺着铜匠来。裴魁山第一个站了出来，说我来第一个骂，骂了许多难听的话，这也是对张一曼的报复。然而铜匠依然不满意，要求剪掉一曼的头发。这时铁男又站出来，拿着剪刀对着众人说，“谁敢剪一下试试”这时特派员的手下拿着枪对着铁男，叫他让开，铁男不让，特派员的手下直接开枪了，不过没有打中。这时的铁男终于屈服了，跪在特派员脚下求他别开枪，最终，一曼的头发被剪完。一曼去照镜子时，疯了。
后来所有的事情全部败露，美国人不在资助他们。片尾，学校依然是那个学校，校长依然是那个校长，铁男依然是那个铁男，魁山依然是那个魁山，然而张一曼，却自杀了。
那么问题来了，谁是这一切的怂恿者？校长，他一次一次的退让，妥协，要求别人拖鞋，所谓顾全大局，做大事不拘小节，事没做成，人毁了一堆。
这部电影暴露出每个人的人性，每个人内心的另一面，只是自己看不见。所有人都走向了自己的反面。 校长的一句“办大事者不拘小节！”毁了多少人。 裴魁山的一句“你就是一个婊子”毁了张一曼的自心。 周铁男的一句“你知道子弹从脸上飞过的感觉是什么吗？我以前不比你横吗？但是有用吗？我告诉你当你被抢指着头的时候你就知道怎么做人了。”毁了自己的勇敢 张一曼的一句“别踩我的花”别人毁了她的所有。 铜匠的一句“有教无类”改变了他自己，他变得面目全非，不在是以前的那个铜匠，好单纯，好不做作。 每个人曾经都有自我，但都被自己抛弃。
然而还有一个人没说，对就是校长的女儿，孙佳，似乎每一个人都是孙佳，处于风暴的边缘，没有被波及，所以可以说：“难道你们就永远这样骗下去吗？”似乎有点站着说话不腰疼的感觉。
有一句话说的好“有人心异变，三头五年就是面目全非，也有人心如止水，十万八千里走过，初心不改。”
那么问题来了，到底要不要招惹这帮臭读书的？
